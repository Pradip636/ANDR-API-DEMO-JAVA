# ApiDemo-Java

This is a simple Android application demonstrating how to fetch data from an API and display it in a list.

## Features

*   Fetches a list of cities from a remote API.
*   Displays the cities in a ListView.
*   Uses Retrofit for networking.
*   Uses Glide for image loading.
*   Uses a Basic Authentication Interceptor for authenticating API requests.

## Dependencies

*   [Retrofit](https://square.github.io/retrofit/): A type-safe HTTP client for Android and Java.
*   [Gson](https://github.com/google/gson): A Java library that can be used to convert Java Objects into their JSON representation.
*   [OkHttp](https://square.github.io/okhttp/): An HTTP client for Android, Kotlin, and Java.
*   [Glide](https://github.com/bumptech/glide): A fast and efficient open source media management and image loading framework for Android.
*   [AndroidX AppCompat](https://developer.android.com/jetpack/androidx/releases/appcompat): Provides backward-compatible versions of Android framework APIs.
*   [AndroidX ConstraintLayout](https://developer.android.com/jetpack/androidx/releases/constraintlayout): Allows you to create large and complex layouts with a flat view hierarchy.
*   [Material Components for Android](https://github.com/material-components/material-components-android): Modular and customizable Material Design UI components for Android.
